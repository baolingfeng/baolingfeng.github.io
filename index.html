Research Overview

I am an Associate Professor at Zhejiang University. My research focuses on software engineering, with particular interests in:

  • Program Comprehension and Maintenance
  • Software Security and Vulnerability Analysis
  • Blockchain and Smart Contract
  • Empirical Software Engineering
  • Mining Software Repositories
  • Developer Behavior Analysis

I employ data mining, machine learning, and empirical methods to understand and improve software development practices, with applications in code recommendation, bug detection, vulnerability analysis, and developer productivity enhancement.
